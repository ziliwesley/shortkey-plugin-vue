Vue Keybinding Plugin Demo

Keybinding plugin: [src/plugins/shortkey.ts](src/plugins/shortkey.ts)

## Use case

This plugin is designed to be mainly used for **global keybinding** inside a page/route. (.e.g. a custom editor)
